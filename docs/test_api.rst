==================
Test API Reference
==================

.. module:: libfs

libfs contains multiple tests tha are run with `CMocka <https://cmocka.org/>`_, and for
convenience, you can include the ``tests/fs_testutils.h`` header to access some
utility functions to help you write your own tests.
